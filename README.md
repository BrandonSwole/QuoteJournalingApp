# History Learning App — README

This project is an educational mobile application that teaches history through quotes from influential figures across various eras. Users receive daily quotes, explore timelines, save favorites, and reflect through journaling. The app uses XAML-based UI (MAUI/Xamarin) for cross‑platform support.

---

## Daily Quote Delivery

* Displays a curated historical quote each day
* Includes author, era, and optional AI-generated context
* Easily share quotes externally

## Timeline Library

* Browse quotes grouped by eras
* Unlock new eras as you progress
* Designed to feel like a visual journey through history

## Reflection & Journal System

* Users write personal reflections for each quote
* All entries saved and organized by date & era
* Favorites and notes are stored for long‑term learning

## Personalized Settings

Includes:

* Preferred eras
* Daily learning time
* Notification preferences
* AI story style (Narrative/Scholarly/Inspirational)
* App appearance (theme, font size, etc.)

---

## Technologies Used

* **.NET MAUI (or Xamarin.Forms)** — cross‑platform UI
* **Local JSON bundles** — stores quote data
  
---
